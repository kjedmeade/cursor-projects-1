# Cursor project

Created the start of a recipe sharing project using Cursor (https://cursor.com/agents)

**Course used:** The Complete AI Coding Course (2025) - Cursor, Claude Code, Udemy

**Tech Stack:**
* NodeJs
* Cursor
* Supabase
